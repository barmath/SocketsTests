Pra rodar os severs bastam 

No Terminal/CMD rodar esse comando
java ReverseServer 9090

No nome ReverseServer pode substituir pelo nome dos outros server das outras pastas. O 9090 representa a porta 

Pra rodar o client 

No Terminal/CMD rodar esse comando
java ReverseClient localhost 9090

vai conectar na porta do servidor 

ATENCAO : Sempre rodar o servidor primeiro
